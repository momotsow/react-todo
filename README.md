### Prerequisites

In order to run this project you need:
 ## Note:


### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/momotsow/react-todo.git
```

### Install

Install this project with:

```sh
  cd math-magicians
  npm install
```
